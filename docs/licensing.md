## FAQ

**Can I use OpenForge in my commercial product?**
Yes, Apache 2.0 allows commercial use.

**Do I need to open-source my modifications?**
No, Apache 2.0 does not require derivative works to be open source.

**Can I sell a product built with OpenForge?**
Yes.

**What if I modify OpenForge for internal use?**
No obligations. Apache 2.0 only requires notice distribution when redistributing.

**Can I use any model with OpenForge?**
You can use any model that is compatible with OpenVINO IR format, subject to the model's license.
